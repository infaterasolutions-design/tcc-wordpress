Premium Fashion Blog and Magazine theme.

== Description ==

The Combo Closet is a pure PHP custom WordPress theme built with Tailwind CSS, designed for a modern fashion blog. It features a sticky header, slide-in mobile drawer, custom homepage layout, and dynamic grid sections.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Development ==

To build the CSS using Tailwind CLI:
1. `npm install`
2. `npm run dev` (for watch mode) or `npm run build` (for minified output).
